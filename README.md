## Prerequisites
I am using `go-task` to create commands for the build.
Hence, you need to insall `go-task`. You can do it following the [instructions no this location](https://taskfile.dev/installation/)

## Start the cluster
```
task build-it
```

## Destroy the cluster
```
task destroy-it
```
